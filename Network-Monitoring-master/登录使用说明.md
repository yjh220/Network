# 登录注册功能使用说明

## 快速开始

### 1. 安装依赖
```bash
pip install Flask-SQLAlchemy==3.0.5 Flask-Login==0.6.3
```

### 2. 初始化数据库
```bash
python init_db.py
```
执行后会创建默认管理员账户：
- 用户名: `admin`
- 密码: `admin123`

### 3. 启动应用
```bash
python app2.py
```

### 4. 访问系统
打开浏览器访问: http://127.0.0.1:8080

## 功能说明

### 未登录用户
- 欢迎页面，显示系统介绍
- 可以点击"登录"或"注册"按钮

### 已登录用户
- 主页显示所有功能模块卡片：
  - 系统仪表板
  - 网络监控
  - 入侵检测
  - 系统日志
  - 系统设置
  - 用户管理（仅管理员可见）

### 用户角色
- **普通用户**: 可访问仪表板、网络监控、日志页面
- **管理员**: 所有功能 + 用户管理（创建/删除/修改用户）

### 会话管理
- 默认会话时长: 1天
- 勾选"记住我"可延长会话

## 常见问题

### Q: 启动时出现 "ModuleNotFoundError"？
A: 请先安装依赖: `pip install Flask-SQLAlchemy Flask-Login`

### Q: 登录/注册后无法跳转到主页？
A: 运行 `python quick_fix.py` 一键修复

### Q: 登录后提示 "用户名或密码错误"？
A: 请先运行 `python quick_fix.py` 修复数据库

### Q: 忘记管理员密码？
A: 运行 `python quick_fix.py` 会重置为 admin123

### Q: 如何添加新用户？
A: 管理员登录后，点击"用户管理" -> "创建用户"

### Q: 点击登录没有反应？
A: 清除浏览器缓存 (Ctrl+Shift+Delete)，或使用无痕模式

### Q: 想要查看详细的调试信息？
A: 运行 `python test_login.py` 检查登录状态
