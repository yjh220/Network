# Network-Monitoring-master 智御网盾 - 项目说明文档

## 一、项目概述

### 1.1 项目简介
本项目是一个基于**SDN（软件定义网络）**和**人工智能**的网络入侵检测与防御系统（NIDS）。系统采用**多智能体协同**架构，结合机器学习和大语言模型，实现网络流量的实时监控、威胁检测、自动防御和态势可视化。

### 1.2 核心创新点
- **双层攻击检测机制**：随机森林快速初筛 + DeepSeek大模型语义复核
- **多智能体协同调度**：Planner/QoS/Security/Query四大智能体协同工作
- **自然语言意图解析**：支持中文指令转换为网络防御策略
- **SDN动态防御**：基于OpenFlow协议的实时流表下发与蜜罐引流
- **RBAC权限管理**：细粒度的用户角色与权限控制系统

## 三、功能模块详解

### 3.1 网络流量监控模块
**功能描述**：实时捕获和分析网络数据包，提取流量特征

| 子功能 | 技术实现 | 说明 |
|-------|---------|------|
| 数据包捕获 | Scapy + WinPcap/Libpcap | 支持BPF过滤器 |
| 协议解析 | Scapy协议栈 | 以太网→IP→TCP/UDP→应用层 |
| 流量统计 | WebSocket实时推送 | 秒级数据更新 |
| 十六进制查看 | 原始字节展示 | 类Wireshark视图 |

### 3.2 入侵检测模块
**功能描述**：基于机器学习和大模型的威胁识别

| 检测层 | 技术 | 检测能力 |
|-------|-----|---------|
| **第一层** | 随机森林分类器 | DDoS、端口扫描、暴力破解等已知攻击 |
| **第二层** | DeepSeek大模型 | 未知攻击、新型威胁、语义分析 |

**检测类型**：
- DDoS攻击（分布式拒绝服务）
- 端口扫描（侦察行为）
- SQL注入（Web攻击）
- XSS攻击（跨站脚本）
- SSH/FTP暴力破解
- 僵尸网络活动
- Heartbleed漏洞利用

### 3.3 SDN防御执行模块
**功能描述**：通过SDN控制器下发防御策略

| 防御手段 | OpenFlow实现 | 说明 |
|---------|-------------|------|
| IP封禁 | Flow Table | 匹配源IP，output=drop |
| 带宽限速 | Meter Table | 速率限制，Kbps |
| 蜜罐引流 | Group Table | 重定向到蜜罐端口 |
| 端口隔离 | Port Modify | 禁用特定端口 |

### 3.4 多智能体协同模块
**功能描述**：自然语言意图解析与策略生成

```
用户输入: "把IP 192.168.1.100封禁掉"
    ↓
Planner Agent解析 → {"action": "block", "target": "192.168.1.100"}
    ↓
Security Agent执行 → 下发Flow Table到交换机
    ↓
SDN Controller → OpenFlow流表生效
```

### 3.5 用户认证与权限管理模块
**功能描述**：基于RBAC的用户权限控制

| 角色 | 权限范围 |
|-----|---------|
| **管理员** | 所有功能 + 用户管理 + 系统配置 |
| **普通用户** | 查看仪表板、监控、日志 |
| **访客** | 仅查看公开页面 |

**权限维度**：
- 功能级：可访问的页面/模块
- 操作级：增删改查权限
- 数据级：可见的数据范围

### 3.6 态势可视化模块
**功能描述**：实时网络安全态势展示

- 实时流量图表（ECharts）
- 协议分布饼图
- 威胁趋势折线图
- 攻击来源地图
- Top N威胁IP排行

## 五、数据流程

### 5.1 攻击检测流程

```
1. 流量捕获
   Scapy sniff → 原始数据包
   
2. 特征提取
   协议解析 → 40+维特征向量
   
3. 第一层检测
   随机森林 → 快速分类 (已知攻击)
        ↓
   可疑流量
        ↓
4. 第二层检测
   DeepSeek大模型 → 深度分析 (未知威胁)
        ↓
5. 威胁判定
   威胁类型 + 严重等级
   
6. 防御响应
   OpenFlow流表下发 / 蜜罐引流
   
7. 告警通知
   WebSocket推送 + 邮件通知
```

### 5.2 多智能体工作流程

```
用户输入自然语言指令
        ↓
Planner Agent 意图识别
        ↓
    ┌───┴───┬───┐
    ↓       ↓   ↓
QoS Agent Security Agent Query Agent
    ↓       ↓       ↓
    └───┬───┴───┬──┘
        ↓       ↓
   SDN适配器 Ryu控制器
        ↓
   OpenFlow流表下发
```
